Mettre à jour l'application

Récupérez l'archive (.zip) contenant la nouvelle version de l'application.

Copiez l'archive dans le dossier basenaturaliste/install.

Si un dossier nommé old existe, supprimez tout ce qu'il contient.

Si le dossier nommé old n'existe pas, créez-le.

Déplacez le fichier backend.ts et le dossier frontend dans le dossier nommé old

Il ne doit plus y avoir de fichier backend.ts ou de dossier frontend dans le dossier basenaturaliste/install.

Dé-compressez l'archive dans le dossier basenaturaliste/install

Vérifiez qu'un fichier backend.ts et un dossier frontend ont été créés dans le dossier basenaturaliste/install.

Supprimez l'archive.

Re-démarrez l'ordinateur.

Lancez l'application et cliquez sur Ctrl+F5.

C'est terminé, vous pouvez utiliser l'application.
